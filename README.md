# TB-downloader
贴吧高清图批量下载器。使用场景：收集贴吧分享的图片素材（表情包、壁纸等）、下载贴吧汉化漫画等等

## 简单说明

1. 下载的是高清原图，不是贴子中的经过压缩后的图。
2. 有任何问题可以在 Issues 里反馈。
